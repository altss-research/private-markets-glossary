# Fund Economics

## Management Fee
Annual fee charged by a fund manager to cover operating expenses, typically calculated as a percentage of committed or invested capital.

## Carried Interest
Performance-based compensation allocated to the fund manager, usually expressed as a percentage of profits above a defined hurdle.

## Hurdle Rate
Minimum return threshold that must be achieved before carried interest is earned.

## Clawback
Provision requiring the fund manager to return excess carried interest if overall fund performance falls below agreed levels.
